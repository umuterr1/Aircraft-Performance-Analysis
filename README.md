# Flight_Data_Processing.ipynb
This script performs a detailed descent analysis to compare the performance characteristics of the real aircraft, the reference model, and the new Flight Management System (FMS).
The comparison evaluates distance, fuel consumption, and thrust setting parameters.
# HORAIRE_json_gen.ipynb & HORAIRE_json_gen_MCL.ipynb
This script automates the generation of JSON input files for Verification and Validation (V&V) activities. 
It computes maximum cruise and maximum climb speeds and automatically generates appropriate scenarios for a selected aircraft model.
# Mr_SESAR_study.ipynb
This script generates multiple climb simulations across various optimization strategies and flight conditions to analyze the climb performance 
of the selected aircraft model and identify the conditions that yield the best performance.
